# Hydrophones Microcontroller

Control settings of ADs Chip
Collecting data bytes with GPIO from ADC chip and send to UART output buffer.

Data Pins:
Pins D0 - D13 on ADC map to PC0 - PC13 on Micro

Other Control & Misc Pins:
CS:PB0
RD:PB1
WR:PB2
CONVST:PB4
SHDN:PB5
ALLON:PB8
EOC:PA9
EOLC:PB7
SWO:PB3
CLK_CNTRL:PB10
CLK:PA8
LED:PB12
